# ⚡️ create-reactwasm-app

Create Emscripten Web Assembly App with React Support from CLI

## 🌀 Install
```
yarn global add create-reactwasm-app
```
or
```
npm install -g create-reactwasm-app
```

## 🔌 Usage
```
create-reactwasm-app <app-name>
```
